# 🛠️ ASSI.CORE v1.0 — Стальное Ядро

### ФИЛОСОФИЯ (АП ЧЁМ БАЗАР)
*   **Максимум контента, минимум настроек**: Навеяно логикой «Эгеи» Ильи Бирмана.
*   **Unix-way**: Всё есть файл.
*   **Патрег — наш Бох**, остальное — суета (привет `vizavi.net`).
*   **Чистая сталь**: Код написан с нуля на нативном PHP. Полное соответствие цветовой дифференциации штанов.

### 🚀 ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ (ПОТРОХА)
*   **Вес**: < 200 KB.
*   **Скорость**: 0.0010 - 0.0030 сек (быстрее пули).
*   **Совместимость**: PHP 5.6 - 8.4+.
*   **БД**: MySQL / MariaDB (PDO).

### ⚡ БЫСТРЫЙ СТАРТ (ИНСТАЛЛ)
1. Распакуй архив в корень сайта.
2. Запусти `http://твой_домен/install.php`.
3. **ВНИМАНИЕ**: Немедленно удали `install.php` после установки!
4. Вход в админку: `/?route=auth` (Дефолт: `admin` / `admin`).

### ⚙️ ПУЛЬТ УПРАВЛЕНИЯ (PULSE CONTROL)
Всё управление в `/core/core.php`. Чтобы отключить модуль, закомментируй строку в массиве `$modules` символом `#`. Это мгновенно скроет раздел и даст 404.

### 🔒 ГЕРМЕТИЧНОСТЬ (SECURITY) & RESET
*   Прямой доступ к папкам закрыт через `.htaccess`.
*   Исполнение PHP в `/storage/` заблокировано наглухо.
*   Сессии защищены фингерпринтом (IP + User-Agent).
*   **СБРОС ПАРОЛЯ**: Если забыл — просто **УДАЛИ** `/core/pass.php`. Пароль станет `admin`.
